# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-03

### Added
- Product catalog management (CRUD, image upload, categorization)
- AI-powered listing generation and optimization (OpenAI)
- Multi-marketplace publishing pipeline
- User authentication with bcrypt password hashing
- Shopping cart and order management
- Brand profile management with voice settings
- Content library for reusable marketing assets
- Social media content scheduling and analytics
- Team management with role-based access
- Marketing campaign tracking
- Style profiles and digital wardrobe (consumer-facing)
- AI stylist chat (consumer-facing)
- Stripe subscription billing (stub)
- Expo/React Native mobile app with Expo Router
- Express API backend deployed on Vercel (serverless)
- Supabase PostgreSQL database with Drizzle ORM
- EAS Build configuration for Google Play Store
- Privacy policy
